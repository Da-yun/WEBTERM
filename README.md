# WEBTERM

### 웹프로그래밍 텀프로젝트 기록
### 외국인을 위한 한국음식문화 홍보 사이트
### 영화 '오징어 게임' 모티브로 제작하였고 흥미 유발을 위해 여러 게임을 제공함
### 캐릭터 및 여러 요소는 svg로 직접 그리거나 무료 제공파일을 다운받아 사용함


# 기능
